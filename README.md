# Todo App Mobile

Projetinho de Todolist simples para aprendizado de mobile com React Native.

## Instalação

```bash
$ npm install
```

## Rodando aplicação

```bash
# development
$ npx expo start

# need tunnel
$ npx expo start --tunnel
```

## Tecnologias utilizadas
- [React Native](https://reactnative.dev/))
- [Expo](https://expo.dev/)
- [Typescript](https://www.typescriptlang.org/)
